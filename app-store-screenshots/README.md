## App Store Connect Upload Instructions

1. Log in to App Store Connect
2. Go to your app > App Store tab
3. Scroll to "App Previews and Screenshots"
4. Select "6.5" Display"
5. Drag and drop screenshots in the order listed above
6. Add localized captions for each screenshot (optional but recommended)
